~ Quick Start Guide ~

Navigate to the web page from where you intend to extract data. You can use the in-built browser to load and navigate web pages. Make sure that you use the mouse for all click actions, instead of 'enter/return' key press of your keyboard.

Once you have reached the page which contains the data to be extracted, click the Start Config toolbar button. Now you can start configuring WebScraper by selecting the data items to capture.

Click on any data item in the page to capture it. WebScraper will intelligently identify patterns of data occurring within a page. The preview of captured data will be displayed in Captured Data Preview pane below. You may capture Text, URLs, Email addresses and images from web pages using the WebScraper.

Once you have finished selecting all the necessary data to be captured, click the Stop Config toolbar button. You may now optionally save the configuration in case you need to run it later. Click the Save toolbar button to save the configuration as an XML file.

At this stage you have finished configuring WebScraper. WebScraper has all the necessary information it need to automatically capture pattern based data from multiple web pages.

Click the Start Mine toolbar button to actually start mining the data. To know more, please ask me!
